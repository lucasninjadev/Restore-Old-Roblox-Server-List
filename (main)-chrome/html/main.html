Auto-Search

Automatically searches for servers the second the panel is loaded.

Faster Searching

Searches for servers faster.

WARNING:
Enabling this almost gurantees that you will not get more than 5 player images per server, however if you are already getting 5 player images per server with this setting disabled, enabling this is recommended.

Servers per page

The amount of servers that are displayed per page.
Default 10, Max 250.

Ascending Player Count

Servers will order from smallest amount of players to largest.

Search Player Username

User that the extension will search for in the servers.

IMPORTANT:
If searching isn't working, it's likely that they are either intentionally not being shown by roblox, or that they were not found in the search.
